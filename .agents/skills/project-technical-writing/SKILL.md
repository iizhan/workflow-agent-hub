# Project Technical Writing

Use this skill when text is part of the deliverable.

## Rules

- Write in Chinese unless the target document is already English.
- Keep text short, factual, and decision-oriented.
- Explain what changed, why it changed, and what remains risky.
- Keep product terms consistent across docs, UI copy, and workflows.
- Update `需求分析` when the writing captures a new durable project decision.
