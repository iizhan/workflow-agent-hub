### 📋 需求摘要

**目标**：PMS 管理后台 Web 登录页面视觉与交互效果优化

### 🎯 方案要点

| 维度 | 优化方向 |
|------|----------|
| **背景** | CSS 渐变（紫→蓝）+ 浮动几何装饰，暗黑模式独立配色 |
| **卡片** | 毛玻璃质感的响应式卡片，`backdrop-filter` 半透明，12px 圆角 |
| **动效** | 卡片入场、表单项 stagger 出现、输入框聚焦、验证码刷新、错误 shake |
| **暗黑模式** | 深色渐变背景 + 暗色毛玻璃卡片 |
| **响应式** | `min(420px, 92vw)` + `768px/480px` 两档断点 |
| **错误提示** | 带动画的表单错误 + 顶部横幅错误通知 |
| **零依赖** | 纯 CSS/SCSS 实现，无新增库 |

### 🔍 范围边界

- **做**：UI 升级、动效、背景、暗黑、响应式、错误提示
- **不做**：OAuth/SSO、MFA、注册页、找回密码、国际化
